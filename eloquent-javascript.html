Exercices from eloquentjavascript.net

Hi,

I read the first 6 chapters of the eloquentjavascript.net website and each time there was an exercise, I wrote the code myself without looking at the proposed solution. Finally I compared my code with the solution and upgraded my code if needed.

I enjoyed keeping a trace of this journey on github.

I was not really convinced by the chapter 8 about object-oriented programming and specially the short explanation about prototypes, so I switched to the book "Secrets of the Javascript Ninja". If you are a JavaScript beginner like me, reading the part 2 "Apprentice training" related to functions, closures and objects will make it clearer.

Hereunder when you click on a chapter, the list of the related exercices are displayed. If you click on the exercice, you see the result of running the code.
